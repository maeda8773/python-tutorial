### 2. Python インタプリタを使う

**学んだこと**

- Python をインストールすると、対話型インタプリタが使える
- ターミナルで `python` と入力すると起動する
- `quit()` を入力すると終了できる
- インタプリタは **対話モード（interactive mode）** で動作する

**プロンプト**

- 一次プロンプト：`>>>`
- 継続行（二次プロンプト）：`...`

### 3. 形式ばらない Python の紹介

#### コメント

- コメントは `#` から始まり、行末までがコメントになる

```python
2 + 2  # 4
```

---

#### 数値演算

- 除算は浮動小数点数を返す
- `//` は整数除算、`%` は剰余を返す

```python
17 / 3     # 5.666666666666667
17 // 3    # 5
17 % 3     # 2
```

---

#### 変数

- 等号 `=` は値の代入に使う

```python
width = 20
height = 5 * 9
```

- 対話モードでは、直前の結果は `_` に代入される

---

#### 文字列

- クォート内でクォートを使う場合はエスケープが必要

```python
'doesn\'t'
```

- `print()` を使うと、エスケープ文字や改行が展開される

```python
s = 'First line.\nSecond line.'
print(s)
```

---

#### 複数行文字列

- 三連引用符 `""" """` や `''' '''` を使う

```python
print("""\
Usage: thingy [OPTIONS]
  -h
  -H hostname
""")
```

---

#### 文字列操作

- インデックスやスライスで文字を取得できる

```python
word = 'Python'
word[0]      # 'P'
word[0:2]    # 'Py'
```

- `len()` で文字列の長さを取得

```python
len('supercalifragilisticexpialidocious')  # 34
```

---

#### リスト

- リストは複数の値をまとめて管理できる

```python
squares = [1, 4, 9, 16, 25]
```

- `append()` で要素を追加

```python
cubes = [1, 8, 27, 64, 125]
cubes.append(216)
```

- リストは参照渡しされる

```python
rgb = ["Red", "Green", "Blue"]
rgba = rgb
id(rgb) == id(rgba)  # True
```
