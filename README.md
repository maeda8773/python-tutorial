## 学習ログ

- 2025/12/24
  **2. Python インタプリタを使う**

- Python をインストール
- ターミナルで `python` と入力するとインタプリタが起動する
- `quit()` を入力すると終了できる
- インタプリタは対話モード（interactive mode）で動作
- ユーザにコマンド入力を促す
- 一次プロンプト：`>>>`
- 継続行の二次プロンプト：`...`
